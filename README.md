# Movies

## Features
- Robust - returns last movie
- Archive - returns 10 last movies

## Dependencies
- Docker

## Running this server 

```bash
$ docker-compose up
```
> [!IMPORTANT]
> Make sure to run this before: docker run -p 8011:80 finout/finout-movies:latest
